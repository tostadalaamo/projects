Calculator in Python

This project is a calculator made on Python that lets you do basic operations, and also letting you restar everytime you finish one, so you can do another one.

This part of the code lets you do all the operations like adding, taking, multiplying, dividing, etc. It also has a bonus part to respond to 0 divided by Zero, responding to it with a "none".

Here is the main cycle of the code, where you input 2 numbers and the type of operation you want to do, and after it does it, it asks if u want to do another operation after you finish the first one.
